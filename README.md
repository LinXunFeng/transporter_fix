## 使用

### 开发版

安装依赖

```shell
# 纯净方式
pip install requests
pip install tqdm

# 懒人方式
pip install -r requirements.txt
```

运行

```shell
git clone git@github.com:LinXunFeng/transporter_fix.git

cd transporter_fix

python main.py
```



### 懒人版

直接下载 [transporter_fix](https://github.com/LinXunFeng/transporter_fix/releases) 后双击运行即可。
